## 🎨 새로운 기능 사용법

### 1. 색상 변경하기
```
1. "펜" 도구 선택
2. 원하는 색상 클릭 (8가지)
3. 그리기 시작!
```

### 2. 지우개 사용
```
1. "지우개" 버튼 클릭
2. 굵기 조절
3. 지우고 싶은 부분 드래그
```

### 3. 학습지 사진 업로드
```
1. 파란색 업로드 영역 클릭
   또는
   이미지를 드래그하여 놓기
2. 배경으로 설정된 학습지 위에 그리기
```

### 4. 전체 지우기
```
"전체 지우기" 버튼 클릭
→ 모든 드로잉 삭제 (배경 이미지는 유지)
```
